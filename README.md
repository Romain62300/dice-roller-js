# 🎲 Lanceur de dés – JavaScript

Petit projet personnel réalisé en **HTML, CSS et JavaScript**.

Il s’agit d’un lanceur de dés virtuel permettant de choisir le nombre de faces, de lancer le dé et de consulter l’historique des résultats.

## 🚀 Fonctionnalités
- Choix du nombre de faces (d4, d6, d8, d10, d12, d20, d100)
- Lancement du dé avec génération aléatoire
- Affichage du résultat
- Historique des lancers (limité)
- Bouton de réinitialisation

## 🛠️ Technologies utilisées
- HTML
- CSS
- JavaScript (vanilla)

## 📂 Structure du projet
- `index.html` : structure de la page
- `style.css` : mise en forme
- `script.js` : logique JavaScript
